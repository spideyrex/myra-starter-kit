# Feature Roadmap — Competing with Filament v4
# Gap analysis of what we're missing vs Filament v4

===============================================
HIGH IMPACT — MUST HAVE
===============================================

## 1. Inline Table Editing
Filament lets users edit data directly in table cells without opening a form.
- ToggleColumn — flip a boolean (e.g., status) right in the table row
- SelectColumn — change a dropdown value inline (e.g., role, status)
- TextInputColumn — edit text inline (e.g., order quantity)
- CheckboxColumn — inline checkbox toggle

## 2. Infolist / Show Pages (Read-Only Detail Views)
Filament has a full "Infolist" builder for structured read-only views.
We only have edit forms, no dedicated show/detail pages for resources.
- A ShowPage component with entries (TextEntry, ImageEntry, BadgeEntry, etc.)
- Reuses the same Section/Grid/Tabs layouts from FormFields
- Replaces building ad-hoc detail cards manually

## 3. Relation Managers
Manage HasMany / BelongsToMany / MorphMany relationships inline on a resource page.
E.g., viewing a User and seeing their Orders, Posts, or Permissions in tabs below — with full CRUD.

## 4. Conditional Field Visibility
visible() / hidden() on fields based on other field values.
E.g., show "Company Name" only when type === 'business'.
Our FormField has no reactive visibility system.

## 5. Table Row Grouping & Summarizers
- Grouping — group rows by a column (e.g., group orders by status)
- Collapsible groups
- Summarizers — Sum, Average, Count, Range at the bottom of columns
  (e.g., total revenue, average order value)

## 6. Drag-and-Drop Row Reordering
Reorder table rows via drag-and-drop with sortable.js.
Saves to a sort_order column. Essential for menus, priorities, ordered lists.

## 7. Dashboard Widget System
Our dashboard is hardcoded. Filament has a widget registry where you can:
- Create reusable stat/chart/table widgets
- Arrange them in a responsive grid
- Different dashboards per role
- Collapsible widgets with filter schemas

===============================================
MEDIUM IMPACT — DIFFERENTIATORS
===============================================

## 8. Advanced Filter Types
- QueryBuilder filter — nested AND/OR condition groups (like Notion filters)
- TernaryFilter — Yes/No/All for nullable booleans
- Date range filter — from/to date filtering
- Filter layout options — above table, in a slide-over, in a popover

## 9. Missing Field Types
- TagsInput — tag management with suggestions, reorder
- KeyValue — table-based key-value pair editing (great for metadata/JSON)
- Slider — numeric range selection
- CheckboxList — multiple checkboxes with search + bulk select/deselect
- MarkdownEditor — markdown with preview (we have RichEditor but not Markdown)
- TimePicker — time-only input

## 10. Table Column Manager
Users can show/hide and reorder columns.
We have toggleableColumns but no drag-to-reorder or persistent preferences (localStorage/DB).

## 11. Sticky Table Header
Keep the header row visible when scrolling a long table.
Simple CSS but very useful for large datasets.

## 12. Real-Time Notifications (WebSocket)
We have database notifications + bell icon, but no live push via WebSocket.
Filament integrates with Laravel Echo/Reverb for instant broadcast notifications.

## 13. Multi-Tenancy
Team/organization scoping — auto-scope queries to current tenant,
tenant switcher, tenant registration. Major for SaaS use cases.

## 14. CLI Generators / Scaffolding
Filament has make:filament-resource, make:filament-page, make:filament-widget.
We have none. A `php artisan make:admin-resource User` command that generates
Controller, Page, Schema would be huge for DX.

===============================================
NICE TO HAVE — POLISH
===============================================

## 15. Record URL (Clickable Table Rows)
Click anywhere on a row to navigate to its detail/edit page,
not just via the actions dropdown.

## 16. Table Poll / Auto-Refresh
Periodic auto-refresh of table data (e.g., every 30s for a real-time dashboard).

## 17. Responsive Column Hiding
Hide certain columns on mobile breakpoints (e.g., hide "Created At" on phones).

## 18. Column Tooltips
Hover over a truncated cell to see the full value in a tooltip.

## 19. Nested Resources
Parent/child URL structure: /projects/{project}/tasks with auto breadcrumbs.

## 20. Vertical Tabs
Tab orientation option — horizontal (current) or vertical sidebar style.

## 21. Striped Table Rows
Alternating row colors for readability.

## 22. Export Enhancements
- XLSX support (not just CSV)
- Column selection modal before export
- Export with current filters applied

## 23. Field Slots
Customizable slots around each form field
(before label, after label, before content, after content).

## 24. Rate-Limited Actions
Throttle destructive actions per user/IP to prevent abuse.
